Treehouse Project 8 Case Study

Live Demo

The Project

I learned AJAX in this unit and the project involved using Promises to acquire the information about random people (representing faux employees) from an external server and using Object Literal destructuring to display it to the page. Each employee had information displayed in a grid of "cards" which will focus on a chosen employees and display their information. Treehouse supplied the syntax for the promises and the destructuring and some base html.

The Notes

The Promise code supplied by Treehouse had a gotcha, which I fixed by saving the data results to a variable called employees. Which allowed me to pass it to the next function to write it to the page.

fetch(urlAPI)
.then(response => response.json())
.then(data => employees = data.results)
.then(displayEmployees)
.catch(err => console.log(err))

We implemented a modal which selects the closest employee card to the user click, to display and had to add navigation controls to the modal to display the next employees like an image gallery. The modal had to have the functionality to navigate through the list of employees using arrow buttons and start over once the user got to end. For example start on employee 1 and to be able to navigate back to employee 1 with the arrows. We had used a plugin to achieve this earlier in the course but realized I wrote my own simplified version of it for this project! Very cool.

function plusSlides(n) {
if(index === 11){
index = 0;
showSlides(index += n);
} else {
showSlides(index += n)
} }

There was a simple search bar which I re-used code from an earlier project.

input.addEventListener("keyup", (e) => {
let search = input.value.toLowerCase();
for (i = 0; i < cards.length; i++){
let name = cards[i].querySelector('h2').textContent.toLowerCase();
if (name.includes(search)) {
cards[i].style.display = '';
} else {
cards[i].style.display = 'none';
}}});

The Summary

This project was a good introduction to a develepors bread and butter which is receiving information from a server asynchronously and then formatting it into HTML to dynamically display on the page. My high point of confidence for the Treehouse Front-End Tech Degree and quickest time of completion.
