Treehouse Project 6 Case Study

Live Demo

The Project

The goal for this project was to use basic Javascript to create a Hangman-style game in which the players guesses the letters of a randomly chosen phrase. If the user chooses the wrong letter, one "life" of five is taken. At the end the button should reset the game with a message based on if the player has won or lost and reset the game with a new phrase.

The Notes

The application begins on the click of the start button which then will grab a random phrase from an array using a variable as index. That function then uses the spread "operator" to spread the phrase into and array of letters and spaces which is then returned.

const getRandomPhraseArray = phraseArr => {
let randomKey = Math.floor(Math.random() * 5);
let randomPhrase = phraseArr[randomKey];
letters = [...randomPhrase]
};

The next step was passing in letters to the next function which loops over the spread phrase and appends the only the letters to the DOM. This got me a little trouble with a local variable. I had no clue wht closures were (still don't) and had my first expreience with scope so to solve I used a global variable.

The Summary

This was my favourite Javascript project from Treehouse. The most important thing in this project was flow control. Watching the program get the phrase, then append the empty letter-boxes to the screen, was a huge confidence booster. Once I saw how the pieces were connected it was easy to understand(ish). Some things I could have done better were modulating some code into helper functions and using less global variables with closures.
